# Swatnet-Build

### How to install Dependencies

```
sudo apt install libssh-dev -y
```

### How to run Swatnet

# Windows:
```
.\swatnet.exe -p <port> -bp <bot_port> -pw <bot_pw>
```

# Linux:
```
./swatnet -p <port> -bp <bot_port> -pw <bot_pw>
```


### How to run the Bot Client

# Linux 
```
./client <ip> <port> <bot_pw>
```
